# 📊 Learning Analytics Dashboard

A full-stack Learning Analytics Dashboard built with **FastAPI + React (Vite) + TailwindCSS** that helps users track, analyze, and improve their study consistency and productivity.

---

## 🚀 Overview

This project allows users to log daily learning sessions and visualize performance using intelligent analytics metrics such as streaks, velocity, consistency, and topic distribution.

It is designed with a modern, clean UI and a scalable backend architecture.

---

## 🛠 Tech Stack

### Backend
- FastAPI
- SQLAlchemy ORM
- PostgreSQL / SQLite
- JWT Authentication
- Pydantic Schemas

### Frontend
- React (Vite)
- TailwindCSS
- Recharts (Data Visualization)
- Framer Motion (Animations)
- React Hot Toast (Notifications)

## 📊 Analytics Logic

- **Weekly Streak** → Consecutive active learning weeks
- **Learning Velocity** → Average weekly hours (last 4 weeks)
- **Consistency Score** → Active study days in last 30 days
- **Trend Analysis** → Weekly aggregated learning hours
- **Topic Breakdown** → Total hours grouped by topic
- **Goal Progress** → Monthly completion percentage
